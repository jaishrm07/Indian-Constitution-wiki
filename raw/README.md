# Raw Sources

This directory is the immutable evidence layer for the project.

Use it for:

- official PDFs
- downloaded judgments
- clipped articles
- extracted text from PDFs
- source-specific working notes

Suggested subdirectories:

- `inbox/`: newly captured source material waiting to be processed
- `official/`: official PDFs and institutional documents
- `extracted/`: text extracted from PDFs or portals
- `notes/`: source-by-source processing notes

Rules:

- do not rewrite the source itself
- do not treat this layer as publishable interpretation
- if a source is important enough to influence the site, summarize it in `knowledge-vault/` and then normalize it into `src/content/`
